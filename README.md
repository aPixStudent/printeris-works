# printeris.works — Mājas lapas apraksts

## Par lapu kopumā

Roberts 3D Print ir neliels, individuāli vadīts 3D printēšanas studijas bizness.
Mājas lapa ir e-veikals ar informācijas lapām, veidota WordPress + WooCommerce.
Valoda: galvenokārt angļu, kontaktu lapa latviešu valodā.

**URL:** https://printeris.works
**Platforma:** WordPress 7.0, WooCommerce, Kadence tēma
**E-pasts:** info@printeris.works
